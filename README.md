# AI Chat App

A simple AI chat app built with Next.js and Hugging Face.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Create a `.env` file with your Hugging Face token:
   ```
   HF_TOKEN=your_token_here
   ```

## Run

```bash
npm run dev
```

Then open http://localhost:3000 in your browser.
